# LilyPaddlerPlus v1.2.0


## Added Features
- Fake values/directions on the O2 bar and the seaglide minimap. (Suggested by @Skippytbm on Nexus Mods!)
- Wiggly GUI icons on areas like the toolbar, inventory menu, and build menu.
- Messing with depth of field (as long as it's enabled in the game's settings).  

## Added Config
Added toggles for new features.  

## Tweaks/Bug fixes  
- Fixed an issue where while having RandomizedBinds enabled, using the scroll wheel to switch hotbar slots could have unpredictable behavior.
- Massive code overhaul, so the mod will probably perform a little better.
